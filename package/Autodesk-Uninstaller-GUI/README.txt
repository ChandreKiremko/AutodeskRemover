AUTODESK UNINSTALLER TOOL (GUI VERSION) - README
===================================================

This tool provides a user-friendly graphical interface to uninstall Autodesk 
products from your system.

== IMPORTANT REQUIREMENTS ==

1. Windows 10 or Windows 11
2. Administrator privileges (essential)
3. Internet browser (Chrome, Firefox, Edge, etc.)

== FILES INCLUDED ==

1. Run-Autodesk-Uninstaller.bat - Batch file to run the application with administrator privileges
2. Autodesk-Uninstaller-Tool.exe - The main application executable

== HOW TO USE ==

1. Right-click on "Run-Autodesk-Uninstaller.bat"
2. Select "Run as administrator"
3. The application will open in your default web browser
4. Select the Autodesk products you want to uninstall
5. Click "Uninstall Selected" and follow the on-screen instructions

== WHAT THIS TOOL DOES ==

1. Shows a list of all installed Autodesk products with a clean graphical interface
2. Lets you select which products to uninstall
3. Provides options to delete the C:\Autodesk folder and restart the computer
4. Shows real-time logs of the uninstallation process
5. Handles all the complex uninstallation steps automatically

== TROUBLESHOOTING ==

- If you see a Windows security warning, click "More info" and then "Run anyway"
- If the browser doesn't open automatically, try running the executable directly as administrator
- If no products appear in the list, make sure you're running as administrator
- Some Autodesk products may require multiple uninstallation attempts due to dependencies
